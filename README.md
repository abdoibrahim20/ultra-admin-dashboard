# Ultra Admin Dashboard 🚀

A modern, responsive, and interactive admin dashboard for managing sales, orders, and customers. Built with **HTML, CSS, and JavaScript**, featuring charts, tables, filters, pagination, and dark/light mode.

---

## 🌟 Features

- **Responsive Layout** – Works on desktop and tablet screens.
- **Sidebar Navigation** – Easy access to Dashboard, Orders, and Customers.
- **Stats Cards** – Display key metrics like Sales, Orders, Customers, Revenue.
- **Interactive Table** – Filter by status, search, and pagination.
- **Chart Visualization** – Line chart showing sales trends.
- **Dark/Light Mode** – Switch theme for comfort.
- **Clean, Modular Code** – Easy to extend and maintain.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Charts:** Chart.js CDN
- **Icons & UI:** Custom CSS (ready to integrate FontAwesome if needed)
